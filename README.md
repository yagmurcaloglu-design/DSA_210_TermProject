# DSA_210_TermProject
Title:
Market Reactions to Global Events: A Case Study Analysis
Proposal:
Financial markets are often affected by major global events such as wars and pandemics. Sudden events can increase uncertainty and lead to noticeable changes in market behavior. The aim of this project is to analyze how financial markets react to selected global events and to identify common patterns in these reactions.
The dataset will be collected from publicly available sources such as Yahoo Finance using Python. The analysis will focus on a limited number of major events, including the COVID-19 pandemic and the Russia-Ukraine War. For each event, data will be collected for a short time window, specifically 30 days before and 30 days after the event. The dataset will include daily values of the S&P 500 index (SPY) and the volatility index (VIX), which represent market performance and risk levels.Key metrics such as returns, volatility, and drawdowns will be analyzed around selected crisis periods.
In the next stage, a machine learning model will be developed to predict the likelihood of a market downturn based on historical patterns and economic indicators. The goal of this project is to provide a clear and data-driven understanding of how global events influence financial markets. This can contribute to a better understanding of financial risk and decision-making under uncertainty.
